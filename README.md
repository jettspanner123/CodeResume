## 🧠 The Idea

Most resume tools are either locked behind paywalls, require accounts, or spit out ugly PDFs. This is none of that.

The resume lives in a **single HTML file**. Every piece of text has `contenteditable="true"` on it — meaning you click directly on the resume in your browser and type. No forms, no editors, no friction. When you're done, hit print and save as PDF.

---

## 🚀 Getting Started

> No installs. No terminal. No build step. Just a browser.

### Steps

**1️⃣ Open the file**
```
Open Uddeshya_Singh_Resume.html in any browser
Chrome / Safari / Firefox / Edge — all work
```

**2️⃣ Click and edit**
```
Click on any text directly on the resume
Name, job titles, bullet points, dates, skills — all editable
A dashed outline appears when you hover over editable fields
```

**3️⃣ Adjust font size**
```
Use the floating  +  /  −  buttons (bottom-right corner)
Scale text up or down until everything fits on one page
The indicator shows current scale as a percentage
```

**4️⃣ Export to PDF**
```
Press Cmd+P  (Mac)  or  Ctrl+P  (Windows)
Set destination → Save as PDF
Hit Save 🎉
```

## ✨ Features

| Feature | Description |
|---|---|
| 🖊️ **In-browser editing** | Every text field is `contenteditable` — click and type, no code needed |
| 🔡 **Font scale controls** | Floating +/− buttons scale all text proportionally |
| 🖨️ **Print-ready PDF** | UI controls and edit outlines are hidden in print output |
| 📦 **Zero dependencies** | Pure HTML, CSS, and ~20 lines of vanilla JS |
| 🌐 **Works offline** | No internet connection required after opening the file |
| 💾 **Persistent scale** | Font scale preference is saved to `localStorage` |
| 📐 **Standard sizing** | Styled for 8.5×11in (US Letter) with 0.5in margins |

## 💡 Tips & Tricks

> 🗂️ **Save your edits** — After editing in the browser, use `File → Save As` to overwrite the HTML file so changes persist on next open.

> 📏 **Fitting one page** — If content spills onto a second page, tap the **−** button to shrink the font scale until everything fits cleanly.

> 🖨️ **Best PDF quality** — Use **Chrome** for printing. It produces the sharpest, most accurate PDF output from HTML.

> 🎨 **Customising styles** — All styles are in the `<style>` block at the top of the HTML file. Fonts, spacing, and colors are easy to tweak.
